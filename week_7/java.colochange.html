Alert Box


It is used to provide a warning message to users. It has only one 'OK' button to continue and select the next task.


Confirm Box


It is widely used for taking the opinion from the user on the specific option. It includes two buttons, which are OK and Cancel.


Prompt Box


The prompt dialog box is used when it is required to pop-up a text box for getting the user input. Thus, it enables interaction with the user.
